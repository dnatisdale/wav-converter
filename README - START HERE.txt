BATCH AUDIO CONVERTER - WINDOWS START GUIDE

EASIEST WAY:
1. Double-click: Start Batch Audio Converter Hidden.vbs
2. Wait a moment.
3. Your browser should open to:
   http://localhost:8080

WHAT THIS DOES:
- It starts the local web server for the PWA.
- It hides the black command window.
- The app runs from your own laptop using localhost.

IMPORTANT:
- Keep the app folder where it is after unzipping.
- The hidden launcher still depends on Python being installed.
- If nothing opens, use the visible launcher:
  Start Batch Audio Converter.bat
  That version shows messages so you can spot errors.

HOW TO STOP THE HIDDEN VERSION:
Option 1:
- Close the browser tab.
- Then open Task Manager and end python.exe or py.exe if it is still running.

Option 2:
- Reboot your computer.

BEST PRACTICE:
- Use the HIDDEN VBS launcher for normal everyday use.
- Use the BAT launcher only if you need to troubleshoot.
